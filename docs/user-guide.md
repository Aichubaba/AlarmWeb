# Руководство пользователя

## О системе

**BlazorApp2** — веб-приложение для мониторинга сетевых устройств (узлов), управления инцидентами (тикетами), просмотра отчётов и схем оборудования. Доступно два типа учётных записей: **Администратор** и **Пользователь**.

---

## Авторизация

Страница входа: `/login`

| Роль      | Логин  | Пароль     |
|-----------|--------|------------|
| Администратор | admin | admin123 |
| Пользователь  | user  | user123   |

После входа система выдаёт cookie-сессию. При попытке перейти на защищённую страницу без авторизации выполняется перенаправление на `/login`.

---

## Навигация

Боковое меню содержит следующие разделы:

| Раздел          | Маршрут          | Описание                                  |
|-----------------|------------------|-------------------------------------------|
| Главная         | `/`              | Панель мониторинга (дашборд)              |
| Последний отчёт | `/lastreport`    | Сводка последних отчётов по всем узлам    |
| Тикеты          | `/tickets`       | Управление инцидентами                    |
| Оборудование    | `/equipment`     | Таблица оборудования                       |
| Узлы            | `/nodes`         | Таблица узлов                              |
| Поиск схем      | `/searchschemes` | Просмотр и загрузка схем узлов            |

---

## Главная страница (`/`)

Дашборд с общей статистикой:

- **Карточки**:
  - Последний отчёт — количество записей в последнем выгруженном отчёте
  - Количество узлов — общее число узлов в системе
  - Количество тикетов — открытые / закрытые
- **Таблица «Статус»** — сводка по состояниям (доступен, ошибки, недоступен и т.д.)
- **Топ-5 проблемных узлов** — узлы с наибольшим числом открытых тикетов. Клик по строке открывает страницу тикетов с фильтром по узлу.

---

## Последний отчёт (`/lastreport`)

Сводка последних полученных отчётов для каждого узла. Система автоматически отслеживает появление новых номеров отчётов.

- Поиск/фильтрация: по названию, ПВН, FD, местоположению
- Сортировка: по доступности (Available) и ошибкам (Errors)
- Постраничная навигация (20 записей на страницу)
- Клик по строке открывает модальное окно с детальной информацией об отчёте:
  - Доступность (зелёный/красный индикатор)
  - Основная информация (ID, модель, серийный номер)
  - Статус: температура, ошибки, очередь
  - Координаты (широта, долгота)
  - Время работы / простоя

## Тикеты (`/tickets`)

Страница управления инцидентами.

### Фильтры поиска

- ID тикета
- FD узла
- ID узла
- Тип ошибки
- Статус (открыт, закрыт, в работе, ожидание, приостановлен, отменён)

### Колонки таблицы

ID, FD, Название узла, Группа, Тип ошибки, Очередь, Статус, Температура, Доступность, Ошибки, Создан, Обновлён.

Столбцы сортируются кликом по заголовку.

### Действия

- **Просмотр тикета** — клик по строке открывает модальное окно с деталями и историей изменений
- **Закрытие тикета** (админ) — кнопка в модальном окне
- **Массовое закрытие** (админ) — выбор нескольких тикетов через чекбоксы и кнопка «Закрыть выбранные» внизу таблицы
- **Изменение статуса** (админ) — кнопки с подтверждением в модальном окне:
  0 — Открыта
  1 — Закрыта
  2 — В работе
  3 — Ожидание
  4 — Приостановлена
  5 — Отменена

### Автообновление

Таблица автоматически обновляется каждые 30 секунд.

### Модальное окно тикета

- Доступность (индикатор)
- Основная информация (ID узла, модель, местоположение, группа)
- Время работы / простоя
- Статус и описание ошибки
- Лента истории изменений (дата, действие, описание, исполнитель)
- Кнопки смены статуса (только для администратора)

---

## Оборудование (`/equipment`)

Таблица со сводкой по оборудованию. Поиск по FD. Отображает:

- ПВН, FD, Рубеж, Оборудование, Тип, БУ, IP-адрес, Сеть, Шлюз, Маска

---

## Узлы (`/nodes`)

Таблица всех узлов. Поиск по названию. Отображает:

- Группа, ПВН, Название, Местоположение, Модель, FD, IP-адрес FD

---

## Поиск схем (`/searchschemes`)

Галерея загруженных схем узлов.

- Фильтр: по FD или ПВН
- Карточка схемы: номер схемы, ПВН, название узла, FD, местоположение, превью изображения
- Клик по превью открывает полноэкранный просмотр изображения
- **Добавление схемы** (только админ):
  - Кнопка «Добавить схему» в правом верхнем углу
  - Выбрать узел из выпадающего списка
  - Загрузить изображение (макс. 10 МБ)

---

## Выход из системы

Кнопка «Выйти» в правом верхнем углу любой страницы (рядом с именем пользователя).
